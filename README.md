# DeskCopy

This project is a desktop and mobile Android application written in Kotlin Multiplatform.
It serves as a localhost tool so that a user can copy text or links from a mobile device on the desktop app.

## Showcase

User sets the ip address of the local desktop he wants to send data to.
The desktop (server) listens to incoming data.
Desktop app can automatically open a url in the browser.
The mobile and desktop app can also share data in real time using web sockets.
